# Security NFR Documentation

## Обзор
Данная директория содержит документацию по требованиям безопасности (Security Non-Functional Requirements) для SecDev Course Project.

## Структура файлов

- **`NFR.md`** - Основная таблица с 10 Security NFR
- **`NFR_BDD.md`** - BDD сценарии в формате Gherkin для проверки NFR
- **`NFR_TRACEABILITY.md`** - Матрица трассировки NFR ↔ User Stories/Tasks
- **`README.md`** - Данный файл с описанием структуры

## Как использовать

### 1. Просмотр NFR
Откройте `NFR.md` для просмотра всех требований безопасности с метриками и порогами.

### 2. Проверка через BDD
Используйте сценарии из `NFR_BDD.md` для написания автоматических тестов.

### 3. Планирование разработки
Используйте `NFR_TRACEABILITY.md` для планирования задач и приоритизации.

### 4. Автоматическая проверка
Запустите скрипт проверки NFR:
```bash
python scripts/check_nfr.py
```

## Интеграция с CI/CD

NFR проверки автоматически выполняются в CI/CD пайплайне:
- Проверка формата ошибок (NFR-02)
- Проверка Rate Limiting (NFR-07)
- Проверка производительности (NFR-03)
- Проверка уязвимостей зависимостей (NFR-04)

## Метрики и мониторинг

Все NFR имеют измеримые метрики:
- **Производительность**: p95 ≤ 300ms
- **Безопасность**: ≤ 7 дней на устранение уязвимостей
- **Доступность**: Rate limiting ≤ 5 запросов/мин
- **Соответствие**: RFC7807 для всех ошибок
